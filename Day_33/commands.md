## DOCKER HANDLING COMMANDS

* `sudo docker ps -a` shows all active and inactive containers, 
* `sudo docker images` - shows downloaded images
* `sudo docker-compose up -d` runs dockers from compose.yml file in detached mode
* `sudo docker exec -it <container_name> bash / sh ` - interactive,terminal,bash/shell
* `sudo docker rm <conatiner_name>` - remove container
* `sudo docker rmi <image_name>` - remove image
