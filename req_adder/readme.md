## requrements.txt file autocompleter

### purpose of life
- program scans typed directory for Python files and 
- add requirements file to directories with project files
- program is based on Python pipreqs module
- program scans only 1st level of child directories 
- (first-child or nth-child(1) ;)

### how to use
- run the script
- type path to scan
- type old **filename** to remove (or just hit enter if there is no any)
- and make yourself some coffe , scanning and generating req file
- per one directory takes usually about 10-15 secs, slow, but 
- work is going on and you have coffe break :) ☕

_*`future release will support multiprocessing`*_  
_*`so it will be probably much faster (depends how many cores you have`*_
