# insta follower

## purpose
- login, handles with popups, select your profile, click your followings
- choose one, go to chosen one, click his/her followers,
- scrolls down/up, follow from 4 his/her followers

## how to prepare
* you need four files `main.py, instafollower.py, constants.py, .env`
* fill them (with love xD) correct data
* .env contains `INSTA_LOGIN=instaLogin@yaho.com INSTA_PASSWORD=instaPassword CHROME_DRIVER_PATH=C:\path\myBot\`
* constants.py contains `INSTA_URL="https://www.instagram.com/"
FAVOURITE_PERSON1="accName"
FAVOURITE_ACCOUNT_HREF1="hrefAccName"
YT_URL="https://youtube.com"
GM_URL="https://mail.google.com"`
* use Chrome, make first run from cmd `"C:\Program Files"\Google\Chrome\Application\chrome.exe --user-data-dir="C:\path\myBot"`

## Enjoy :)
